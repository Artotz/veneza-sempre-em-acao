# AGENTS

## Checklist obrigatorio (i18n)
- Todo texto exibido na UI deve usar `t("...")`.
- Ao adicionar/alterar textos, registre a chave em `src/i18n/pt-BR.ts`.
- Use chaves em formato objeto, por exemplo `ui.botao_salvar` ou `labels.page.login`.
- Prefira chaves existentes; se criar novas, mantenha consistencia de idioma (pt-BR) e acentos corretos.
- Nunca finalizar uma task com erros de TypeScript.
